# Booking Service

Сервис для бронирования переговорок с поддержкой ролей администратора и пользователя.

## Технологии

- **Язык:** Go 1.25.0
- **База данных:** PostgreSQL
- **Библиотеки:** JWT, pgx, Swagger, Testcontainers, Mock
- **Инструменты:** Docker Compose, golang-migrate, Makefile

## Структура проекта

- `cmd/`: Точка входа приложения (main.go)
- `internal/`: Бизнес-логика (handler, service, repository)
- `migrations/`: SQL-миграции базы данных
- `tests/`: Интеграционные тесты
- `unit-test/`: Юнит-тесты и покрытие
- `api/`: OpenAPI спецификация (api.yaml)

## Реализованные функции

- **Администратор:**
  - Создание переговорок
  - Создание расписания доступности (дни недели, время)
  - Просмотр списка всех броней с пагинацией

- **Пользователь:**
  - Просмотр списка переговорок
  - Просмотр доступных слотов по переговорке и дате
  - Создание и отмена брони
  - Просмотр своих броней

- **Общие:**
  - JWT-аутентификация через `/dummyLogin`
  - Регистрация и авторизация по email/паролю (`/register`, `/login`)
  - Автоматическое формирование слотов на основе расписания (30-минутные интервалы)
  - Валидация бизнес-правил
  - Swagger UI для API документации (доступен по `/swagger/index.html`)
  - Тестирование: юнит-тесты, интеграционные тесты для сценариев создания и отмены брони

## Запуск проекта

   ```
   make up
   ```
   Это выполнит линтинг, тесты, сборку и запуск сервиса с базой данных, а также применит миграции.

Сервис будет доступен на `http://localhost:8080`.

Для остановки: `make stop` или `make down`.
